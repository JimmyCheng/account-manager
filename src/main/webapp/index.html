Spring REST

Search API

  • Get All the accounts http://localhost:8080/account-manager/api/v1/account
  • Get the specific account(id=1)http://localhost:8080/account-manager/api/v1/account/1

Modification API

  • Create Account：http://localhost:8080/account-manager/api/v1/account method=Post, consumes=JSON
  • Update Account(id=1001)：http://localhost:8080/account-manager/api/v1/account method=Put, consumes=JSON
  • Delete Account(id=1001)：http://localhost:8080/account-manager/api/v1/account/1 method=Delete, consumes=JSON
